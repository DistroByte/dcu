GAME DEVELOPMENT

What is a development pipeline?

The game development pipeline is an industry term used to describe the process of building a video game from concept to completion.

A game is continuously evolving, and ideas which sound appealing in theory may not work so well in reality. Therefore, the pipeline is not necessarily a linear process and must be flexible enough to factor revisions and changes.

The 3 stages of game development

While pipelines vary between projects and studios, the process is fairly similar whether you’re working on a AAA, indie or mobile game.

Video game development is typically divided into 3 stages:
Pre-production, Production & Launch.
